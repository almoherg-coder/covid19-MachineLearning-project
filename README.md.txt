# COVID-19 Prediction Project

## Overview
This project predicts whether a patient is COVID-19 positive based on symptoms and risk factors using Machine Learning. It includes exploratory data analysis, visualizations, and testing multiple algorithms.

## Dataset
Source: Kaggle â€“ [Symptoms and COVID Presence](https://www.kaggle.com/datasets/hemanthhari/symptoms-and-covid-presence)

## Features
- Breathing Problem, Fever, Dry Cough, Sore throat, Running Nose, Asthma, etc.

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Models Tested
- Logistic Regression
- Decision Tree
- Random Forest


## Results
- Accuracy, Precision, Recall for each model
- Feature importance visualizations

## How to Run
1. Clone the repository
2. Install required libraries: `pip install -r requirements.txt`
3. Open the Jupyter Notebook `COVID.ipynb` to explore analysis and predictions
